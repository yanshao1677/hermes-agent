# 阶段3：主循环前收集临时上下文并在工具循环中复用

## 触发条件

已得到本轮真实消息列表和稳定系统提示词，且尚未进入模型调用与工具循环。只要存在插件钩子或外部记忆管理器，就在主循环前各收集一次临时上下文。

## 输入字段

- `original_user_message`：字符串，清洗后的原始用户输入；用于外部记忆查询，避免技能或临时注入污染查询词。
- `messages`：真实历史消息列表；传给插件时使用副本。
- `conversation_history`：旧历史是否存在，用于计算 `is_first_turn`。
- `session_id`、`model`、`platform`、`sender_id`：插件钩子元数据。
- `memory_manager`：可选外部记忆管理器。

## 判定规则

1. 插件上下文在主循环前通过 `pre_llm_call` 钩子收集一次。
2. 插件返回值为字典且存在 `context` 字段时，取 `str(context)`；返回值为非空字符串时，直接取该字符串。
3. 多个插件上下文以两个换行拼接为一个本轮插件上下文字符串。
4. 插件钩子抛异常时，记录警告并把插件上下文视为空字符串。
5. 外部记忆 prefetch 在主循环前收集一次，查询词使用 `original_user_message`；若该字段不是字符串，则用空字符串。
6. 外部记忆 prefetch 抛异常时，动作是跳过，缓存为空字符串。
7. 进入工具循环后，不再重新调用插件钩子和外部记忆 prefetch；每次构造 API 请求副本时复用已缓存的两个字符串。

## 状态读写位置

- 写入本轮局部变量：`plugin_user_context`、`ext_prefetch_cache`。
- 不写入真实消息列表，不写入系统提示词快照，不写入会话数据库。
- 跨工具循环复用依赖局部变量仍在同一次对话运行栈内；跨轮次不复用。

## 正常路径

1. 调用插件的 `pre_llm_call` 钩子，传入当前会话标识、原始用户输入、真实消息副本、是否首轮、模型、平台和发送者。
2. 遍历插件返回值，提取字典的 `context` 或非空字符串。
3. 拼接插件上下文。
4. 若存在外部记忆管理器，用原始用户输入调用 `prefetch_all()`。
5. 把结果保存为本轮缓存。
6. 进入主工具循环；每轮 API 请求构造时读取同一份缓存。

## 分支路径

- 插件返回空列表或所有结果为空 → 插件上下文为空，不影响主流程。
- 插件返回字典但无 `context` → 忽略该项。
- 插件返回非字符串、非目标字典 → 忽略该项。
- 插件钩子异常 → 记录警告，继续。
- 无外部记忆管理器 → 外部记忆上下文为空。
- 外部记忆 prefetch 异常 → 吞掉异常，继续。

## 失败处理

插件失败不阻断；外部记忆失败不阻断。证据不足：源码能证明失败会跳过并继续，但没有证据说明是否有可观测指标或重试策略；实现时若需要运维可观测性，应增加失败计数或 tracing。

## 幂等性 / 一致性约束

本阶段只提供“单轮一致性”：同一用户轮次内，不论模型调用多少次工具，模型看到的临时召回上下文都来自主循环前那一次收集。它不能保证外部记忆服务的实时最新状态，也不保证插件结果可重放。

## 代码骨架

```python
def collect_turn_context(hooks, memory_manager, state, messages, original_user_message, had_history):
    plugin_context = ""
    try:
        results = hooks.invoke(
            "pre_llm_call",
            session_id=state.session_id,
            user_message=original_user_message,
            conversation_history=list(messages),
            is_first_turn=not bool(had_history),
            model=state.model,
            platform=state.platform or "",
            sender_id=state.sender_id or "",
        )
        parts = []
        for item in results:
            if isinstance(item, dict) and item.get("context"):
                parts.append(str(item["context"]))
            elif isinstance(item, str) and item.strip():
                parts.append(item)
        plugin_context = "\n\n".join(parts)
    except Exception:
        plugin_context = ""

    memory_context = ""
    if memory_manager:
        try:
            query = original_user_message if isinstance(original_user_message, str) else ""
            memory_context = memory_manager.prefetch_all(query) or ""
        except Exception:
            memory_context = ""
    return plugin_context, memory_context
```

## 最小验证清单

- 单轮内有 3 次 API 调用时，外部记忆 `prefetch_all()` 只被调用 1 次。
- 插件返回 `{"context": "A"}` 与字符串 `"B"` 时，缓存为 `A\n\nB`。
- 插件异常时，仍进入模型主循环。
- 外部记忆异常时，仍进入模型主循环。
- 查询词使用清洗后的原始用户输入，而不是已附加技能内容的 API 用户消息。
