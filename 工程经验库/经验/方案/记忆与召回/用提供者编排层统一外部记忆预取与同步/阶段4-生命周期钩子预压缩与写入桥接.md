# 阶段4：生命周期钩子预压缩与写入桥接

## 触发条件

| 钩子 | 触发点 |
|------|--------|
| `on_pre_compress(messages)` | 本地 flush 之后、compressor 之前 |
| `on_memory_write(action, target, content)` | builtin memory 工具 add/replace 成功后 |
| `on_delegation(task, result, ...)` | 子智能体委托完成 |
| `on_session_end(messages)` | CLI exit / gateway session 结束 |
| `shutdown_all()` | 会话真正结束 teardown |

## 输入字段

- `on_pre_compress`：即将被压缩的 messages 列表。
- `on_memory_write`：`action` in (`add`,`replace`)，`target`，`content`；**跳过** builtin provider 自身。
- `on_delegation`：子任务描述与结果摘要、`child_session_id`。
- `on_session_end`：完整 messages 快照。

## 判定规则

**on_pre_compress**

1. 遍历 non-builtin providers，`provider.on_pre_compress(messages)`。
2. 非空 strip 结果以 `\n\n` join，返回给压缩器（供摘要 prompt 附加）；单 provider 失败 → debug skip。
3. 调用位置：`_compress_context()` 内 `flush_memories` 之后。

**on_memory_write**

1. 主循环 memory 工具路径：action in `("add","replace")` 时调用。
2. 遍历 providers，`name=="builtin"` → continue。
3. 通知 external 本地偏好变更。

**on_delegation / on_turn_start / on_session_end**

- 编排层遍历调用；失败 debug/warning，不 raise。

**shutdown_all**

- 逆序 `provider.shutdown()`；在 session 真正结束路径调用，**非**每 turn。

## 状态读写位置

- 读写：provider 外部存储。
- 不写：本地 snapshot（仍由本地方案阶段 6 刷新）。

## 正常路径

1. 用户 memory add → external 收到 on_memory_write。
2. 上下文压缩 → flush → on_pre_compress → compress。
3. 用户 /reset → on_session_end → shutdown_all。

## 分支路径

- 无 external provider → 钩子仍可被调用但仅 builtin（on_memory_write 跳过 builtin）。
- 钩子异常 → 压缩/主流程继续。

## 失败处理

全部 non-fatal。证据不足：on_pre_compress 返回文本是否一定进入压缩摘要 prompt 取决于 compressor 调用方。

## 幂等性 / 一致性约束

on_memory_write 不触发本地 snapshot 刷新。external 与 local 可能短暂不一致直到 reload。

## 代码骨架

```python
def on_local_memory_write(manager, action, target, content):
    if action not in ("add", "replace"):
        return
    for p in manager.providers:
        if p.name == "builtin":
            continue
        try:
            p.on_memory_write(action, target, content)
        except Exception:
            pass
```

## 最小验证清单

- compress 路径在 flush 之后调用 on_pre_compress。
- memory remove 不触发 on_memory_write。
- 每 turn 结束不调用 shutdown_all。
