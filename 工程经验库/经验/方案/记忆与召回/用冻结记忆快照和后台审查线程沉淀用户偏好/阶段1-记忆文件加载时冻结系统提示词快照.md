# 阶段1：记忆文件加载时冻结系统提示词快照

## 触发条件

智能体初始化且 `skip_memory=False`，并且 `memory_enabled=True` 或 `user_profile_enabled=True` 至少一个为真。

## 输入字段

| 字段 | 类型 | 默认 | 来源 |
|------|------|------|------|
| `memory_enabled` | bool | false | 配置 `memory.memory_enabled` |
| `user_profile_enabled` | bool | false | 配置 `memory.user_profile_enabled` |
| `memory_char_limit` | int | 2200 | 配置 `memory.memory_char_limit` |
| `user_char_limit` | int | 1375 | 配置 `memory.user_char_limit` |
| 偏好文件路径 | path | `{home}/memories/MEMORY.md` | profile 作用域 home |
| 画像文件路径 | path | `{home}/memories/USER.md` | 同上 |

## 判定规则

1. 仅当 `memory_enabled OR user_profile_enabled` 时实例化 `MemoryStore`。
2. 调用 `load_from_disk()`：创建目录、`read_file` 两路径、条目 `strip` 后过滤空串。
3. 去重：`list(dict.fromkeys(entries))`，保留第一次出现顺序。
4. 渲染块：header 含 target 标签 + `[pct% — current/limit chars]` + 46 个 `═` 分隔线 + `ENTRY_DELIMITER.join(entries)`。
5. 快照写入：`_system_prompt_snapshot = {"memory": render(...), "user": render(...)}`。
6. `format_for_system_prompt(target)` 只返回 `_system_prompt_snapshot[target]`；空串返回 `None`。
7. 初始化异常：外层 `try/except` 吞掉，`MemoryStore` 不创建，智能体继续启动。

## 状态读写位置

- 读：磁盘 MEMORY.md / USER.md。
- 写：内存 `memory_entries`、`user_entries`、`_system_prompt_snapshot`。
- 不写：磁盘（本阶段只读）。

## 正常路径

1. 读取配置项。
2. 构造 `MemoryStore(memory_char_limit, user_char_limit)`。
3. `load_from_disk()` → 读文件 → 去重 → 渲染 → 捕获快照。
4. 后续 `_build_system_prompt()` 在 `memory_enabled` 时 append memory 块，在 `user_profile_enabled` 时 append user 块。

## 分支路径

- 文件不存在 → 对应 entries 为 `[]`，snapshot 对应键为 `""`，系统提示词不注入该块。
- 文件读 OSError → `_read_file` 返回 `[]`。
- 仅 `user_profile_enabled` → 只注入 user 块；仅 `memory_enabled` → 只注入 memory 块。
- `skip_memory=True` → 跳过整个阶段。

## 失败处理

- 初始化 `try/except Exception: pass` → 记忆不可用，无用户可见错误。
- 证据不足：单文件损坏而另一文件正常时，是否部分加载——源码按文件独立 `_read_file`，可部分加载。

## 幂等性 / 一致性约束

快照在 `load_from_disk()` 完成瞬间固定。同一会话内多次调用 `format_for_system_prompt()` 结果相同。刷新快照的唯一入口是再次 `load_from_disk()`（见阶段 6）。

## 代码骨架

```python
ENTRY_DELIMITER = "\n§\n"

def load_from_disk(store):
    store.memory_entries = dedupe(read_entries(store.path("memory")))
    store.user_entries = dedupe(read_entries(store.path("user")))
    store.snapshot = {
        "memory": render_block("memory", store.memory_entries, store.memory_char_limit),
        "user": render_block("user", store.user_entries, store.user_char_limit),
    }

def format_for_system_prompt(store, target):
    block = store.snapshot.get(target, "")
    return block if block else None
```

## 最小验证清单

- 重复条目加载后只保留第一次。
- `memory_enabled=False` 且 `user_profile_enabled=False` 时不创建 store。
- 空文件时 `format_for_system_prompt("memory")` 返回 `None`。
- 加载后 snapshot 字符数与 `ENTRY_DELIMITER.join(entries)` 长度一致。
