# 阶段7：主响应交付后后台审查记忆和技能

## 触发条件

`run_conversation()` 回合收尾，同时满足：

1. `final_response` 非空。
2. `interrupted == False`。
3. `_should_review_memory OR _should_review_skills` 为真。

## 输入字段

| 字段 | 说明 |
|------|------|
| `messages_snapshot` | `list(messages)` 浅拷贝 |
| `review_memory` | 来自阶段 4 |
| `review_skills` | `iters_since_skill >= skill_nudge_interval` 于回合结束判定 |
| `skill_nudge_interval` | 默认 10，配置 `skills.creation_nudge_interval` |
| 审查 prompt | `_MEMORY_REVIEW_PROMPT` / `_SKILL_REVIEW_PROMPT` / `_COMBINED_REVIEW_PROMPT` |

## 判定规则

**spawn 条件（已在触发条件）**

**后台线程 `_run_review`**

1. stdout/stderr redirect 到 `/dev/null`。
2. 新建 `AIAgent(model=同主会话, max_iterations=8, quiet_mode=True, platform, provider)`。
3. 注入共享状态：`review_agent._memory_store = self._memory_store`（及 enabled 标志）。
4. 关闭子 agent nudge：`memory_nudge_interval=0`, `skill_nudge_interval=0`。
5. `run_conversation(user_message=prompt, conversation_history=messages_snapshot)`。
6. 扫描子 agent `_session_messages` 中 `role=tool` 且 `success=true` → 提取 memory/skill 操作摘要。
7. 若有 actions → `_safe_print(f"  💾 {summary}")` 及可选 `background_review_callback`。
8. `finally`: `review_agent.close()` 释放 httpx 等资源。

**线程属性**

- `threading.Thread(target=_run_review, daemon=True, name="bg-review")`。
- spawn 外层 try/except 吞掉异常。

**时序**

- 必须在 `final_response` 已写入 result **之后**执行（与 sync_all 同区块，在 return 前）。
- 不修改主 `messages` 或 `final_response`。

## 状态读写位置

- 读：messages 快照、共享 MemoryStore live 状态。
- 写：MEMORY/USER 文件、技能目录（通过子 agent 工具）。
- 不写：主会话 `_cached_system_prompt`、主 messages。

## 正常路径

1. 主循环返回用户可见回复。
2. sync 外部记忆（另一方案）。
3. 若 nudge 标志 → start daemon thread。
4. 子 agent 审查并可选 write memory。
5. 用户看到紧凑 `💾` 摘要（若有写入）。

## 分支路径

- `interrupted=True` → 不 spawn。
- `final_response` 为空 → 不 spawn。
- 仅 skill nudge → 用 skill 审查 prompt。
- 两者皆有 → combined prompt。
- 子 agent 无 successful tool → 无用户可见输出。
- 子 agent 异常 → debug 日志，主流程已完成。

## 失败处理

全程 best-effort。不保证审查完成；不 retry。审查写入不刷新主会话 snapshot（同阶段 2）。

## 幂等性 / 一致性约束

非审计级：同一对话可能多次审查写入；依赖 memory add 去重。主用户已收到响应，审查不影响 SLA。

## 代码骨架

```python
def maybe_review(agent, messages, final_response, interrupted, review_memory, review_skills):
    if not final_response or interrupted or not (review_memory or review_skills):
        return
    prompt = pick_prompt(review_memory, review_skills)

    def run():
        sub = QuietAgent(max_iterations=8, same_model_as=agent)
        sub.memory_store = agent.memory_store
        sub.memory_nudge_interval = 0
        sub.run_conversation(prompt, conversation_history=list(messages))
        notify_successful_writes(sub.session_messages)

    threading.Thread(target=run, daemon=True, name="bg-review").start()
```

## 最小验证清单

- 有 final_response 且未中断时 spawn 不阻塞 return。
- 子 agent `max_iterations=8`。
- 主 `_cached_system_prompt` 在审查 write 后不变。
- 审查线程为 daemon。
